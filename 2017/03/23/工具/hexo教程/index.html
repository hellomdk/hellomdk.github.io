    主题切换

    1. 先在hexo或这github上找到喜欢的主题
    2. 进去thems主题目录
      cd thems
    3. 下载主题
      git clone https://github.com/yelog/hexo-theme-3-hexo.git themes/3-hexo
    4. 修改project根目录的_config.yml文件
      theme: 3-hexo（与主题名称一致）
    5. 生产静态页面hexo g(注意有时需要hexo clean清理下)
    6. 部署到github
