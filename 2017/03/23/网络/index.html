    常用的网络IO模型

    IO模型:

    1. 同步阻塞IO
    2. 同步非阻塞IO
    3. IO多路复用
    4. 信号驱动IO
    5. 异步IO

    概念:

    1. 阻塞|非阻塞：是对于用户线程来说的（是否阻塞等待）
    2. 同步|非同步：是对于IO响应方式来说的（同步即是数据从内核拷贝到用户空间的过程需要阻塞等待）
    3. 内核空间|用户空间：用户线程是没有操作内核空间的权限的，需要调用系统指令操作内核数据
    4. 数据拷贝: 数据从网卡copy到内核空间，再从内核空间copy到用户空间
    5. 磁盘IO|网络IO：（本质上是内存与硬件之间的交互）
      1. 磁盘IO是磁盘与内存之间的输入输出
      2. 网络IO是网卡与内存之间的输入输出
